### 🫂 MLOps 기반 영화 추천 서비스 및 서비스 관리
사용자 측면에서 사용자가 영화 줄거리를 입력하면 해당 줄거리와 유사한 영화를 추천해 주는 서비스를 제공합니다.  
관리자 측면에서는 사용자 서비스의 전반적인 부분을 운영/관리할 수 있는 MLOps 서비스를 제공합니다.
